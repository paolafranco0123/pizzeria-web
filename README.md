# 🍕 La Pizzeria — Sitio Web

Sitio web diseñado y desarrollado para **La Pizzeria**, restaurante ubicado en Rota, Cádiz. Proyecto de portfolio que demuestra habilidades de maquetación web, diseño responsive y experiencia de usuario.

🔗 **Demo en vivo:** [lapizzeriarota.netlify.app](https://lapizzeriarota.netlify.app)

---


## ✨ Características

- Diseño **responsive** adaptado a móvil, tablet y escritorio
- **Menú interactivo** con tabs y sistema de acordeón para categorías
- Sección de **especialidades con fotografías reales** del local
- **Horario semanal** con estados abierto/cerrado
- **Mapa de Google Maps** embebido con la ubicación del local
- Animaciones de entrada con **Intersection Observer API**
- Banner animado con **scroll infinito** de categorías
- Integración con **Instagram** y enlace a Google Maps
- **CSS custom properties** para sistema de diseño coherente
- **SEO básico** con meta tags y estructura semántica HTML5

---

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 (custom properties, grid, flexbox, animaciones)
- JavaScript vanilla (tabs, acordeón, Intersection Observer)
- Google Fonts (Bebas Neue, Nunito)
- Font Awesome 6 (iconos)
- Google Maps Embed API
- Desplegado en **Netlify** con integración continua desde GitHub
